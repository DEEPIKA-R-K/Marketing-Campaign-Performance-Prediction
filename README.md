# 💄 Beauty Brand Campaign Analytics & Profit Prediction

An end-to-end **Data Analytics and Machine Learning** project that analyzes digital marketing campaign performance for **Nykaa, Purplle, and Tira**. The application provides interactive visualizations, business insights, and predictive analytics using **Python, Streamlit, Scikit-learn, Pandas, Matplotlib, and Seaborn**.

---

## 📌 Project Overview

Marketing campaigns generate large volumes of data, making it challenging to evaluate their effectiveness and profitability. This project helps businesses analyze campaign performance, compare brands, and predict future campaign outcomes using Machine Learning.

The application features an interactive Streamlit dashboard that enables users to:

- Analyze campaign performance
- Compare marketing metrics across brands
- Visualize campaign trends
- Predict campaign profit
- Classify campaigns as Profit or Loss

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/your-username/Beauty-Brand-Campaign-Analytics.git
```

Navigate to the project directory

```bash
cd Beauty-Brand-Campaign-Analytics
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the Streamlit application

```bash
streamlit run app.py
```
